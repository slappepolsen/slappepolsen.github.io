# slappepolsen.github.io

**Live site:** https://slappepolsen.github.io/

The site is served from the repo root (`index.html`). The README is only for visitors on GitHub; it does not power the live page.

## If you see “Site not found” on GitHub Pages

1. Open **Settings → Pages** for this repo.
2. Under **Build and deployment → Source**, choose **Deploy from a branch**.
3. Set **Branch** to `main` and folder **`/ (root)`**, then **Save**.
4. Keep this repo **Public** if you use free GitHub Pages for a user site.
5. Under **Custom domain**: if something is listed, either fix DNS for that domain or clear the field to use `https://slappepolsen.github.io` only.
6. Wait a few minutes, then hard-refresh the site (e.g. Cmd+Shift+R).

To force a new publish after fixing settings, push any small commit to `main`.
